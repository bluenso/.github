We build embedded systems, automotive software, IoT platforms, and AI-powered tools — from bare-metal firmware to production backends. 17+ years shipping software that runs on hardware in the real world.

---

### Core expertise

**Embedded & firmware**
Low-level C/C++ development on microcontrollers and microprocessors. Real-time systems with FreeRTOS and bare-metal schedulers. BSP, device drivers, serial protocols (UART, SPI, I2C, RS-232), ADC/GPIO, OTA bootloaders with secure update chains. Targets include ARM Cortex-M (RP2040, i.MX28), ESP32-S3, and Fujitsu 16-bit MCUs.

**Automotive**
Vehicle diagnostics via K-Line (ISO 9141-2), CAN bus, UDS, OBD-II, FMS, and RDL. Digital tachograph download and fleet telemetry. BLE/WiFi gateway development for wireless diagnostic access. Secure firmware with HMAC-SHA256 authentication and FOTA update flows.

**IoT & industrial**
Custom hardware integration with programmable I/O expansion boards (ADC, relay, opto-isolated digital I/O). Multi-protocol communication layers. Smart metering with DLMS/COSEM (IEC 62056), wMBUS, HDLC, AES-128-GCM encryption. Compliance testing tooling against industry standards (UNI TS 11291).

**AI & agents**
Multi-agent AI systems built with LangGraph and LangChain. RAG pipelines, computer vision with YOLO11 and ByteTrack, speech recognition with Faster-Whisper, integration with Claude and GPT-4o APIs. Deployed across domains: financial advisory, job matching, smart home assistants.

**Backend & tooling**
Python/Django backends for device management, remote configuration, and data processing. Multi-tenant architectures, REST APIs, Celery/RabbitMQ task queues, PostgreSQL. Qt5 desktop applications with serial data acquisition, real-time plotting, PDF reporting.
